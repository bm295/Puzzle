# Personas and User Journeys

## Personas

### P1: Organization Admin
Goals:
- Configure organization, roles, and policies.
Pain points:
- Hard to audit changes and permissions.

### P2: Instructor
Goals:
- Build courses quickly and assess learners fairly.
Pain points:
- Manual grading overhead and poor visibility into at-risk learners.

### P3: Learner
Goals:
- Complete coursework on time with clear guidance.
Pain points:
- Unclear deadlines, fragmented feedback, low motivation.

## Core journeys

### J1: Admin setup
1. Create organization
2. Invite instructor and learners
3. Configure defaults (grading scale, timezone)

### J2: Instructor launches course
1. Create course
2. Add modules/lessons
3. Create quiz/assignment
4. Publish and enroll learners

### J3: Learner completes course
1. Enroll and access dashboard
2. Complete lessons and submit assessments
3. Review feedback/grades
4. Achieve completion/certificate status

### J4: Instructor intervention
1. Identify at-risk learners
2. Send reminders/announcements
3. Adjust due date or grading rules

## UX principles

- Deadline clarity over feature density
- Progress visibility by default
- Instructor efficiency over clicks
- Accessibility first
